- **Path traversal via absolute EDL paths**: wrong — `ensure_within`
  (`helpers/common.py:28`) resolves and rejects anything outside the footage
  root; validator/exporters/preview all call it.
- **XXE / billion-laughs via FCPXML import**: overstated — `xml.etree` never
  fetches external entities, and bundled expat ≥ 2.4 caps entity expansion;
  FCPXML input is a user's own editor export. Not worth a plan.
- **Transcript overwrite risk in transcribe.py**: wrong — cached-transcript
  early return at `helpers/transcribe.py:109-111`.
- **O(N²) repeated-take scan in pack_transcripts**: wrong — the scan breaks
  at the 45s `repeat_window` boundary (`pack_transcripts.py:62-63`); bounded,
  not quadratic.
- **Missing FCPXML export→import round-trip test**: wrong — exists
  (`tests/test_import_fcpxml.py:67`).
- **update_fcpxml.py untested (claimed 182 lines)**: overstated — it is a
  42-line wrapper over the well-tested `write_fcpxml`.
- **Integer overflow in seconds_to_frames**: not real — Python ints are
  arbitrary precision.
- **`.bak.json` overwritten on repeated `--replace`**: documented
  single-backup behavior (README), by design.
- **Per-cut audio decode in audio_refine**: windowed decode is the design;
  an up-front full decode trades memory for marginal speed. Not worth doing.
- **Timing-logic consolidation across timing.py/common.py/export_fcpxml.py**
  and **render/qa preview dedup**: MED-risk refactors with low benefit;
  overlap is smaller than it looks. Not worth doing.
- **F401 unused-import rule missing**: ruff's default set already includes F.
