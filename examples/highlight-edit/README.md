# Highlight Edit Example

This example shows a short highlight edit built from a longer interview source.
The source media is intentionally omitted; `raw/founder-interview.mp4` is the
expected local path if you want to validate or export the sample EDL.

The EDL demonstrates:

- Non-contiguous source ranges.
- Continuous record timing.
- Beat, quote, reason, and transform metadata.
- Subtitle output path metadata.

Expected result: a 30 second timeline assembled from four selected moments.
